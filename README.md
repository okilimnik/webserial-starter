### Huawei Freebuds commands
Noise cancelling on:
```script
+␄␁␁␁
```
Noise cancelling off:
```script
+␄␁␁␀
```
Look for more commands in the original [article](https://mmk.pw/en/posts/freebuds-4i-proto/)

### Tests
Running tests require headless Chrome installed.
```bash
npm run test
```
